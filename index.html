1. Food Delivery Website using [ DAO Design Pattern]

I developed a food delivery website using the DAO design pattern, JDBC, Servlets, JSP, and deployed it on Apache Tomcat Server with a MySQL database as the backend.

(i) Architecture & Design: Implemented the DAO (Data Access Object) pattern for clean separation of business logic and data persistence.

(ii) Backend: Built with Java Servlets to handle client requests and responses. Used JDBC for database connectivity and CRUD operations. Managed user authentication, restaurant management, menu items, and order processing.

(iii) Frontend: Designed user-friendly interfaces with JSP pages for restaurant listings, menus, cart, and order placement. Implemented dynamic rendering of data from the database.

(iv) Database: Designed and normalized the MySQL database schema with tables for users, restaurants, menu items, and orders. Applied foreign key constraints for referential integrity.

(v) Deployment: Hosted on Apache Tomcat Server for local testing and execution.

Key Features:

  • User Registration and Login Page
  • Browse Restaurants and Menus
  • Add to Cart, Check Out and Place Orders.

Technologies Used:

JDBC Servlets JSP Apache Tomcat Server MYSQL
